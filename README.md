# 부산대학교 전자전기컴퓨터 학술동아리 퓨마스&amp;화이트컴 홈페이지

## 현재 구현된 기능
1. 회원가입 및 로그인, 로그아웃
2. 공지사항 글쓰기 및 상단바 알림
3. 파일 다운로드

## 미구현 기능
1. 자유 게시판 및 스터디 게시판
2. 재무재표
